# Scientific Calculator

This is a **Scientific Calculator** built using Python and Tkinter.

---

## Features

* Basic operations (+, -, *, /)
* Scientific functions (sin, cos, tan, log, sqrt)
* Power calculation (^)
* Dark and Light mode
* Degree / Radian mode
* History of calculations
* Backspace and keyboard support

---

## How to Run

1. Open terminal in project folder
2. Run:

```bash
python main.py
```

---

## Project Files

* `main.py` → Main program (UI)
* `calculator.py` → Calculation logic
